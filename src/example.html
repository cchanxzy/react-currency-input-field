React Currency Input Field

Features:
  • Allows abbreviations eg. 1k = 1,000 2.5m = 2,500,000
  • Prefix option eg. £ or $
  • Can allow/disallow decimals
  • Automatically inserts comma separator
  • Only allows valid numbers
  • Lightweight and simple
